# Privacy Policy for GSC Weekend Filter

**Last updated:** April 2026

GSC Weekend Filter is a Chrome extension that adds a Business Days View to Google Search Console performance reports by excluding weekends and, optionally, holidays from the chart.

## What the extension does

The extension processes chart and report data available in Google Search Console in order to display a filtered Business Days View directly in the user's browser.

## Data the extension handles

The extension may access and process data shown in Google Search Console performance reports while the extension is running on that page. This data is used only to provide the extension's core functionality.

The extension does **not** collect personal identification information, authentication credentials, payment information, health information, personal communications, or location data.

## Local storage

The extension stores user preferences locally in the browser using `chrome.storage.local`.

These preferences may include:
- whether the feature is enabled
- filtering options
- selected country for holiday filtering
- cached holiday dates

This local storage is used only so the extension can remember settings and reduce unnecessary repeated requests.

## Holiday data

If the user enables holiday filtering, the extension may request public holiday data from `https://date.nager.at/` in order to exclude those dates from the chart.

These requests are used only to retrieve holiday dates needed for the feature.

## How data is used

Any data handled by the extension is used only for the extension's single purpose: displaying a Business Days View inside Google Search Console.

The extension does not use user data for advertising, profiling, credit evaluation, or any unrelated purpose.

## Data sharing

The extension does not sell user data.

The extension does not transfer user data to third parties except as technically necessary to retrieve public holiday dates from `https://date.nager.at/` when the user enables holiday filtering.

## Data retention

User preferences stored with `chrome.storage.local` remain in the browser until the user changes them, clears extension storage, or uninstalls the extension.

## Security and scope

The extension is designed to use only the permissions needed for its stated purpose.

## Your choices

You can disable the extension, change its settings, clear the extension's local storage, or uninstall the extension at any time.
